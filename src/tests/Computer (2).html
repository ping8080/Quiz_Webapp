Question No: 1

Which of the following is the primary function of the CPU in a computer?

Options:

  • 1.

    Process instructions

  • 2.

    Store data

  • 3.

    Display graphics

  • 4.

    Manage peripherals

Solution:

    • The CPU (Central Processing Unit) is responsible for processing instructions, which include performing arithmetic and logical operations, and managing data flows between different components of the computer. It is essentially the brain of the computer.

    Incorrect Answers:

    • Store data: Data storage is handled by storage devices like hard drives, SSDs, and RAM, not the CPU.
    • Display graphics: Graphics display is managed by the GPU (Graphics Processing Unit), not the CPU.
    • Manage peripherals: Peripheral management is typically handled by I/O devices and system controllers, not the CPU directly.

Question No: 2

Options:

  • 1.

    Read Access Memory

  • 2.

    Random Access Memory

  • 3.

    Random Allocation Memory

  • 4.

    Read and Access Memory

Solution:

    • RAM (Random Access Memory) is a type of computer memory that can be accessed randomly, meaning any byte of memory can be accessed without touching the preceding bytes. It is used to store data and machine code currently being used by the CPU.

    Incorrect Answers:

    • Read Access Memory: This is incorrect. RAM allows both reading and writing.
    • Random Allocation Memory: This is not a standard term in computing.
    • Read and Access Memory: This is incorrect, as RAM allows both reading and writing.

     

Question No: 3

Which of the following is a non-volatile memory?

Options:

  • 1.

    RAM

  • 2.

    Cache memory

  • 3.

    ROM

  • 4.

    Registers

Solution:

    • ROM (Read-Only Memory) is non-volatile, meaning it retains data even when the computer is powered off. It is used to store firmware and other essential system information.

    Incorrect Answers:

    • RAM: RAM is volatile memory, meaning it loses all data when the power is turned off.
    • Cache memory: Cache memory is volatile and is used to temporarily store frequently accessed data.
    • Registers: Registers are small, fast storage locations inside the CPU, and they are volatile.

Question No: 22

Who is known as the “father of the computer”?

Options:

  • 1.

    Charles Babbage

  • 2.

    Alan Turing

  • 3.

    John von Neumann

  • 4.

    Bill Gates

Solution:

  • The correct answer is a. Charles Babbage.

    Charles Babbage is considered the "father of the computer" for his conceptual design of the analytical engine in the 1830s. His work laid the foundation for the modern computer. While the machine was never fully built in his lifetime, it had all the components of a modern computer, including an arithmetic logic unit, control flow through conditional branching and loops, and memory.

    Distractors:

    1. Alan Turing: Alan Turing was a brilliant mathematician and is regarded as the "father of theoretical computer science" and artificial intelligence. He is famous for his work on the Turing machine, which is a model of computation.
    2. John von Neumann: John von Neumann was a key figure in the development of the stored-program concept, which is a fundamental principle of modern computers, but he is not the "father of the computer."
    3. Bill Gates: Bill Gates is a co-founder of Microsoft and a key figure in the development of personal computing software, but he is not considered the "father of the computer."
